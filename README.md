# Exercicio 1 (parte A) da lista de sub-rotinas  

Faça um algoritmo que leia um número N inteiro e positivo. O algoritmo deverá ter uma sub-rotina que irá calcular
e mostrar o fatorial deste número. O cálculo do fatorial deve ser feito em uma sub-rotina tipo procedimento. Não
passe parâmetros para a sub-rotina, utilize somente variáveis globais.

Desenvolvido utilizado JDK 21 e NetBeans 29
